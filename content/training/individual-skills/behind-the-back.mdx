## 비하인드 더 백 & 비트윈 더 레그란?

크로스오버의 다음 단계로, **등 뒤(비하인드 더 백)**와 **다리 사이(비트윈 더 레그)**로 공을 넘기는 드리블 기술입니다. 수비수가 크로스오버를 읽기 시작하면, 이 기술들로 공의 경로를 더 예측하기 어렵게 만들 수 있습니다.

## 단계별 연습법

### 스텝 1: 비트윈 더 레그 — 제자리 (5분)

1. 한 발을 앞에, 다른 발을 뒤에 놓기 (스탠스 벌리기)
2. 오른손으로 드리블 → **앞에 놓은 다리 사이로** 왼손에 전달
3. 왼손으로 드리블 → 다시 다리 사이로 오른손에 전달
4. **리듬을 유지하면서** 20회 반복
5. 반대 발 앞으로도 동일하게 연습

### 스텝 2: 비하인드 더 백 — 제자리 (5분)

1. 발을 어깨 너비로 벌리고 무릎 구부리기
2. 오른손으로 허리 높이에서 드리블
3. 공을 **등 뒤로 통과**시켜 왼손으로 받기
4. 왼손에서 다시 등 뒤로 오른손에 전달
5. **핵심**: 손목 스냅으로 공을 보내고, 반대 손은 허리 뒤에서 대기
6. 20회 반복

### 스텝 3: 걸어가며 조합 (3분)

1. 코트 끝에서 시작
2. 3~4번 드리블 → **비트윈 더 레그** → 방향 전환
3. 3~4번 드리블 → **비하인드 더 백** → 방향 전환
4. 반대편 끝까지 번갈아 사용하며 이동
5. 돌아올 때는 속도를 약간 높여 반복

### 스텝 4: 콤보 드릴 (2분)

1. 크로스오버 → 비트윈 더 레그 → 비하인드 더 백을 **연속으로** 실행
2. 제자리에서 10세트 반복
3. 세 기술이 물 흐르듯 자연스럽게 연결되는 것이 목표

## 핵심 포인트

- **비트윈 더 레그**: 앞쪽 다리를 넓게 벌려야 공이 지나갈 공간이 생김
- **비하인드 더 백**: 손목 스냅이 핵심 — 팔 전체가 아닌 손목으로 조절
- 두 기술 모두 **시선은 전방**에 — 공을 보지 않고도 할 수 있어야 실전에서 사용 가능
- 화려함보다 **정확성과 속도**에 집중 — 실수 없이 할 수 있을 때만 실전 투입
